# git-push-automation

If you are pushing to a remote repo always like:

1. git pull
2. git add .
3. git commit -m" feat: a lazy commit"
4. git push

### Then this shell script is for you!! 

Just add it to your root repo and run the following:

```
 sh easier_push.sh <commit message> 
```
